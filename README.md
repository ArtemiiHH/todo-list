# ✅ Todo List App

A simple project-based todo app built with vanilla JavaScript and localStorage.

# Features

* Create and delete **projects**
* Add todos with **title, description, due date, priority**
* Mark tasks as **completed** (checkbox + styling)
* Delete individual tasks
* Data persists with **localStorage**

# Structure

* **Controller** – handles logic and events
* **Renderer** – updates DOM
* **Storage** – saves/loads data
* **Todo/Project factories** – create data objects

# Run

```
npm install
npm run dev
```

# Build

```
npm run build
```
